ChitChat

ChitChat is a real-time chat application that allows users to sign up with a username and email, secured through OTP verification. Once authenticated, users can connect and chat instantly with others. The platform supports text messaging as well as media sharing, creating a smooth and engaging communication experience. Built using Express.js, MongoDB, Socket.io, and JWT for authentication, this project demonstrates how to implement secure sign-up flows, real-time messaging, and media handling in a scalable web application.
